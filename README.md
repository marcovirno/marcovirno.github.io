# Marco Virno — Portfolio

Istruzioni rapide per eseguire l'app in locale.

## Run Locally

Prerequisiti: Node.js

1. Installare le dipendenze:

   `npm install`
2. Avviare l'app in modalità sviluppo:

   `npm run dev`
